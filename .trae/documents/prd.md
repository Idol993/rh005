## 1. 产品概述

智慧医院综合管理系统是一套面向医疗机构的全流程数字化平台，覆盖患者挂号、智能分诊、候诊排队、处方审核、检查报告AI解读、住院IoT监测、药房自动发药、出院结算及院长驾驶舱等核心业务场景。
- 解决传统医院流程割裂、排队效率低、用药安全隐患、数据孤岛等痛点，实现全流程智能化闭环
- 目标用户为患者、医护、行政人员及院长，通过四级权限体系保障数据安全与操作合规

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 患者 | 手机号实名注册 | 挂号、查看报告、医保管理、支付结算、健康解读查看 |
| 医护 | 医院内部账号分配 | 诊疗开方、查看患者病历、检查报告审核、住院监测响应 |
| 行政 | 医院内部账号分配 | 药房管理、排班管理、患者导诊、系统配置 |
| 院长 | 系统管理员分配 | 全部数据查看、驾驶舱、运营报表导出、科室筛选 |

### 2.2 功能模块

1. **患者挂号页面**: 症状描述、科室选择、既往病史上传、医保信息绑定、智能医生推荐、自动分诊
2. **候诊排队页面**: 实时叫号队列、预计等待时间、动态队列调整
3. **处方审核页面**: 处方开具、药物相互作用检测、过敏史校验、冲突拦截提示
4. **检查报告页面**: 报告列表、AI异常指标提取、健康解读推送
5. **住院监测页面**: IoT手环生命体征监测、异常实时预警、护士站告警面板
6. **药房发药页面**: 处方调配队列、机械臂发药状态、扫码核对记录
7. **出院结算页面**: 费用明细、医保/自费拆分、床旁支付
8. **院长驾驶舱页面**: 实时数据看板、门诊量/床位周转率/药占比/满意度指标、科室筛选、月报导出

## 3. 核心流程

### 挂号到就诊流程
患者登录 → 输入症状描述 → 上传既往病史 → 绑定医保信息 → 系统智能推荐医生 → 确认挂号 → 进入候诊队列 → 系统动态调整叫号 → 叫号就诊 → 医生开方 → 系统审核药物冲突 → 处方生效 → 药房调配发药

### 住院到出院流程
办理住院 → 佩戴IoT手环 → 实时监测生命体征 → 异常预警护士站 → 医生查房 → 开具检查 → AI解读报告 → 出院结算（医保/自费拆分） → 床旁支付 → 出院

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**: 医疗蓝(#0A6EBD) + 生命绿(#12B886) + 预警红(#FA5252)
- **辅助色**: 深灰(#1A1A2E)背景卡片、浅灰(#F8F9FA)背景
- **按钮风格**: 圆角(8px)、微阴影、hover渐变过渡
- **字体**: 标题使用"DM Sans"（粗体/半粗体），正文使用"DM Sans"（常规/中等），数据数字使用"JetBrains Mono"
- **布局风格**: 左侧导航栏 + 顶部面包屑 + 主内容区卡片式布局
- **图标风格**: 线性图标(Lucide)，统一2px描边

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI要素 |
|----------|----------|--------|
| 登录页 | 角色选择 | 四角色卡片横排、渐变背景、图标标识、hover放大动效 |
| 患者挂号 | 症状输入 | 多行文本域、AI分析加载动画、推荐标签 |
| 患者挂号 | 病史上传 | 拖拽上传区域、文件缩略图、OCR进度条 |
| 患者挂号 | 智能推荐 | 医生卡片列表、忙闲状态指示灯、评分星级 |
| 候诊排队 | 叫号队列 | 大字号当前叫号、排队进度条、时间倒计时 |
| 处方审核 | 冲突拦截 | 红色弹窗拦截、冲突详情、替代药品建议 |
| 检查报告 | AI解读 | 报告PDF预览、异常指标高亮标注、健康解读卡片 |
| 住院监测 | 体征大屏 | 实时折线图、颜色编码状态、异常闪烁动画 |
| 住院监测 | 预警面板 | 红色告警卡片、患者信息、一键响应按钮 |
| 药房发药 | 发药核对 | 机械臂动画、扫码状态步骤条、核对结果 |

| 院长驾驶舱 | 数据看板 | 大数字卡片、趋势迷你图、环形进度图 |

### 4.3 响应式设计

- 桌面端优先（1440px基准），适配1280px/1024px
- 住院监测大屏支持1920x1080全屏模式
- 关键数据卡片在768px以下纵向堆叠
